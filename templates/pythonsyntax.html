Python Syntax & Comments

Execution of Python Syntax

As we know, Python syntax can be executed by writing directly in the Command Line:

>>> print("Hello, India!")
Hello, India!

Or in the Command line by creating a python file on the server, helloindia is the file name with extension .py and run it.

C:\users\saket\python helloindia.py

Python Indentation

Spaces at the beginning of a code line is called indentation.

The indentation in Python is very important.

In Python indentation is used to indicate a block of code.

e.g.

if 10 > 4:
  print("ten is greater than four")

It will give an error if we do not provide the indentation:

e.g.

if 10 > 4:
print("ten is greater than four")

It will give an error because we have not provided the indentation:

e.g.

if 10 > 4:
 print("ten is greater than four") 
if 10 > 4:
        print(""ten is greater than four"") 

We can use spaces in the same block of code.

e.g.

if 10 > 4:
 print("ten is greater than four")
        print("ten is greater than four")

Here python will give an error because number of spaces use in same block of code.

Python Variables
There is no command or keyword to create a variable in python like others languages. When the value is assigned to a variables at the same time variable is created accordingly.

x = 10 # x is a integer type variable created.

y = "Hello, India!" # y is a string type variable created.

Comments

In any programming language comments are used to documentation purpose only. It describes what about the source code is going to be use. Comments start with a #, and Python will render the rest of the line. It explains the logic or a part of it used in the code. Comments are usually helpful to programmer maintaining or enhancing the code when we are no longer around to answer questions about it.

(i) # Symbol is used to a single line comment

However, there is no concept of multiline comments in Python. But it can be achieved by the following ways.

In Python if literals that are not assigned to a variable, we can add a multiline string using triple quotes in our code, and place your comment inside it:

""" here is a comment 2nd line is written 3rd line is written """

# here is a comment
# 2nd line is written
# 3rd line is written

e.g # here is single line comment.
