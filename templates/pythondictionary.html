Python Dictionary

A dictionary is a collection which is unordered, changeable, indexed and does not allow duplicates values. In Python dictionaries are written with curly brackets {}, and they have keys and values.

Create and print a dictionary.
My_dict = {
"Brand_Key": "suzuki_value",
"Model_Key": "newmodel_value",
"Year_Key": 2020
}
print(My_dict)

We can access the items of a dictionary by referring to its key name, inside square brackets. Due to it is unordered we cannot refer to an item by using an index.
My_dict = {
"Brand_Key": "suzuki_value",
"Model_Key": "newmodel_value",
"Year_Key": 2020
}
x = My_dict["Model_Key"]      #Get the value of the "model_Key" key.
print(x)

The get() method that will give the same result.
My_dict = {
"Brand_Key": "suzuki_value",
"Model_Key": "newmodel_value",
"Year_Key": 2020
}
x = My_dict.get("Model_Key") # Get the value of the "model_Key" key.
print(x)

Change the value of a specific item by referring to its key name. Change the "year" to 2018.
My_dict = {
"Brand_Key": "suzuki_value",
"Model_Key": "newmodel_value",
"Year_Key": 2020
}
My_dict["Year_Key"] = 2019
print(My_dict)

My_dict = {
"Brand_Key": "suzuki_value",
"Model_Key": "newmodel_value",
"Year_Key": 2019,
"Year_Key": 2020
}
print(My_dict)   # duplicate values will overwrite existing values.

The update() method change key value.
My_dict = {
"Brand_Key": "suzuki_value",
"Model_Key": "newmodel_value",
"Year_Key": 2019,
}
My_dict.update({"Year_Key ": 2020})
print(My_dict)

Loop Through a Dictionary by using a for loop. the return value are the ''keys'' of the dictionary.
My_dict = {
"Brand_Key": "suzuki_value",
"Model_Key": "newmodel_value",
"Year_Key": 2020
}
for x in My_dict:
print(x)

secondly we can use values() method.
for x in My_dict.values():
print(x)

string, int, boolean, and list data types.
My_dict = {
"brand": "suzuki",
"electric": True,
"year": 2109,
"colors": ["white", "blue", "gray","red"]
}
print(My_dict)
Print the data type of a dictionary.
print(type(My_dict))

Loop Through a Dictionary by using a for loop. The return value are the ''key’s value'' of the dictionary.
My_dict = {
"Brand_Key": "suzuki_value",
"Model_Key": "newmodel_value",
"Year_Key": 2020 }
for x in My_dict:
print(My_dict[x])

Loop through both keys and values, by using the items() method.
My_dict = {
"Brand_Key": "suzuki_value",
"Model_Key": "newmodel_value",
"Year_Key": 2020
}
for x, y in My_dict.items():
print(x, y)

Check if Key Exists- To determine if a specified key is present in a dictionary use the "in" keyword. Check if "model_Key" or ''Brand_Key'' are present in the dictionary.
My_dict = {
"Brand_Key": "suzuki_value",
"Model_Key": "newmodel_value",
"Year_Key": 2020
}
if "Model_Key" in My_dict:
print("Yes, 'searching key' is present in the My_dict dictionary")

Dictionary Length- To determine how many items (key-value pairs) a dictionary has, use the len() function.
My_dict = {
"Brand_Key": "suzuki_value",
"Model_Key": "newmodel_value",
"Year_Key": 2020,
"Date_Key": ""
}
print(len(My_dict))

Adding an item to the dictionary is done by using a new index key and assigning a value to it.
My_dict = {
"Brand_Key": "aifa_value",
"Model_Key": "newmodel_value",
"Year_Key": 2020,

}
My_dict["Date_Key"] = "22-06-2020"
print(My_dict)

The pop() method removes the item with the specified key name.
My_dict = {
"Brand_Key": "aifa_value",
"Model_Key": "newmodel_value",
"Year_Key": 2020,
"Date_Key": "20.06-2020"
}
My_dict.pop("Model_Key")
print(My_dict)

The popitem() method removes the last inserted item (in versions before 3.7, a random item is removed instead).
My_dict = {
"Brand_Key": "aifa_value",
"Model_Key": "newmodel_value",
"Year_Key": 2020,
"Date_Key": "20.06-2020"
}
My_dict.popitem()
print(My_dict)

The del keyword removes the item with the specified key name.
My_dict = {
"Brand_Key": "suzuki_value",
"Model_Key": "newmodel_value",
"Year_Key": 2020,
"Date_Key": "20.06-2020"
}
del My_dict["Model_Key"]
print(My_dict)

The del keyword can also delete the dictionary completely.
My_dict = {
"Brand_Key": "aifa_value",
"Model_Key": "newmodel_value",
"Year_Key": 2020,
"Date_Key": "20.06-2020"
}
del My_dict
print(My_dict)   # this line will cause an error because "My_dict" no longer exists.

The clear() method empties the dictionary:
My_dict = {
"Brand_Key": "aifa_value",
"Model_Key": "newmodel_value",
"Year_Key": 2020,
"Date_Key": "20.06-2020"
}
My_dict.clear()
print(My_dict)

The copy() method copies a dictionary and insert into a new one.
My_dict = {
"Brand_Key": "aifa_value",
"Model_Key": "newmodel_value",
"Year_Key": 2020,
"Date_Key": "20.06-2020"
}
Your_dict = My_dict.copy()
print(Your_dict) # copied dictionary
print(My_dict)   # original dictionary

Or Make a copy of a dictionary with the dict() function.
My_dict = {
"Brand_Key": "aifa_value",
"Model_Key": "newmodel_value",
"Year_Key": 2020,
"Date_Key": "20.06-2020"
}
Your_dict = dict(My_dict)
print(Your_dict)  # copied dictionary
print(My_dict)    # original dictionary

Nested Dictionaries, a dictionary can also contain many dictionaries, this is called nested dictionaries.
Nested_dict = {
"dict_1" : {
"Name" : "Saket",
"Sex" : "Male"
},
"dict_2" : {
"Name" : "Kunal",
"Sex" : "Male"
},
"dict_3" : {
"Name" : "Leena",
"Sex" : "Female"
}
}
print(Nested_dict)

if we want to nest three dictionaries that already exists as dictionaries.
dict_1 ={
"Name" : "Saket",
"Sex" : "Male"
}
dict_2 ={
"Name" : "Kunal",
"Sex" : "Male"
}
dict_3 ={
"Name" : "Leena",
"Sex" : "Female"
}

Nested_dict = {
"dict_1" : dict_1,
"dict_2" : dict_2,
"dict_3" : dict_3
}
print(Nested_dict)

The dict() constructor to make a new dictionary.
My_dict = dict(Brand_Key="aifa_value", Model_Key="newmodel_value", Year_Key=2020)
print(My_dict)
