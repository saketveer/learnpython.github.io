Python Variables

Variables
Variable or scalar is a storage location (identified by a memory address). Variable name associated with symbolic name, which contains some known or unknown quantity of information referred to as a value.

Variable Names
A variable can have a short name (like x and y) or a more than one character (Sex, sex, name, Name, Tot etc.).

Rules for Python variables:
• A variable name must start with a letter or the underscore character
• A variable name cannot start with a number
• A variable name can only contain alpha-numeric characters and underscores (A-z, 0-9, and _ )
• Variable names are case-sensitive (Sex, sex and SEX are three different variables)
e.g.
Legal variable names:
myvar = "Saket"
my_var = "Saket"
_my_var = "Saket"
myVar = " Saket"
MYVAR = " Saket"
myvar3 = " Saket"

e.g.
Illegal variable names:
3myvar = "Saket"
my-var = " Saket"
my var = " Saket"

Creating Variables
Python has no keyword or command for declaring a variable.
A variable is created as like we first assign a value to it.
e.g.
x = 10
y = "saket"
z=”A”
print(x) 
print(y) 
print(z) 

Note:
In Python, Strings are arrays of bytes representing Unicode characters. A string is a collection of one or more characters put in a single quote, double-quote or triple quote. In python there is no character data type, a character is a string of length one. It is represented by str class.

Casting
If we need to specify the data type of a variable, it can be done with casting.
e.g.
x = str(5) # x will be '5'
y = int(5) # y will be 5
z = float(5) # z will be 5.0

We will learn more about data types and casting later in this tutorial.

Single or Double Quotes?
String variables can be declared either by using single or double quotes:
e.g.
x = "saket"
x = 'saket'        # here x is the same as

Case-Sensitive
Variable names are case-sensitive.
e.g.
It will create two variables:
r = 10
R= "Saket"
#both will not overwrite to each other.

Assign Multiple Values
Python allows to assign value to multiple variables in one line:
e.g.
x, y, z = "Guava", "Pineapple", "Mango"
print(x) # x will print Guava
print(y) # y will print Pineapple
print(z) # z will print Mango

Python allows to assign the same value to multiple variables in one line:
e.g.
x = y = z = "Mango"
print(x) 
print(y)
print(z)      # the all three variables x, y & z will print Mango

 

Unpack a Collection of variables
If we have a collections of values in a tuple, list etc. Python allows to extract the values into variables separately. It is called unpacking.
e.g.
Unpacking a list:
_Items_fruits = ["Guava", "Mango", "Orange"]
x, y, z = _Items_fruits
print(x) # x will print Guava
print(y) # y will print Mango
print(z) # z will print Orange

Concatenation in Python
To combine or concatenate both text and a variable, Python uses the + character:

e.g.
x = "excellent"
print("Python is " + x) # the result be "Python is excellent"
We can also use the + character to add a variable to another variable:
e.g.
x = "Python is "
y = "excellent"
z = x + y
print(z) # the result be "Python is excellent"
For numbers, the + character works as a mathematical operator:
e.g.
x = 20
y = 30
print(x + y) # result will be 30
If we try to combine a string and a number, it will give an error.
e.g.
x = 20              # here x is an integer variable
y = "Hello"     # here y is a string variable
print(x + y)   # it will give an error. Because we can't add string and integer

Global Variables
The Variables are created outside of a function is  called Global variables and can be used by everyone, both inside and outside of functions.
e.g.
Create a variable outside of a function, and use it inside the function
x = "excellent"

def testfunc():
print("Python is " + x)
testfunc()    # result will be " Python is excellent " 

Variable inside a function, is local variable and can only be used inside the function. While the variable with the same name out side of a function is a global variable and it will remain with the original value as it was.

 x = "excellent"  # global variable
def testfunc():
  x = "powerful"  # local variable
  print("Python is " + x)  # result will be "Python is powerful" 
testfunc()
print("Python is " + x)  #  result will be "Python is excellent"

The global Keyword
To create a global variable inside a function use the global keyword to create global variable.

def testfunc():
global x
x = "powerful"
testfunc()
print("Python is " + x)  # result will be "Python is powerful"

To change the value of a global variable inside a function, refer to the variable by using the global keyword:
x = "excellent"
def testfunc():
global x
x = "powerful"
testfunc()

print("Python is " + x)  # result will be "Python is powerful"
