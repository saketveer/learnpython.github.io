Python Class/Object

Classes / Objects
Python is an object oriented programming language. Almost everything is an object in python, with its properties and methods. A Class is like an object constructor, or a "blueprint" for creating objects.

Create a Class.
The keyword class is used to create a class:
class Car_Class:
x = 10        # here Car_Class, with a property named x:
print(Car_Class)

Create an Object.
We can use the class name Car_Class to create objects named obj1, and print the value of x:
class Car_Class:
x = 10
obj1 = Car_Class()
print(obj1.x)

The __init__() Function.
All classes have a function called the built-in __init__(), which is always executed when the class is being initiated.
Create a class named Student, use the __init__() function to assign values for name and age.
class Student:
def __init__(self, name, age):
self.name = name
self.age = age
obj1 = Student("Saket", 36)
print(obj1.name)
print(obj1.age)
# The __init__() function is called automatically every time the class is being used to create a new object.

Object Methods.
Methods in objects are functions that belong to the object.
class Student:
def __init__(self, name, age):
self.name = name
self.age = age
def method_func(self):
print("Hi my name is " + self.name,"and age is ", self.age)
obj1 = Student("Saket", 36)
obj1.method_func()

The self Parameter.
It is a reference to the current instance of the class, and is used to access variables that belongs to the class. We can use another name instead of self, it has to be the first parameter of any function in the class.
class Student:
def __init__(myobject, name, age):
myobject.name = name
myobject.age = age
def method_func(pqr):
print("Hi my name is " + pqr.name,"and age is ", pqr.age)
obj1 = Student("Saket", 36)
obj1.method_func()

Modify Object Properties.
class Student:
def __init__(self, name, age):
self.name = name
self.age = age
def method_func(self):
print("Hi my name is " + self.name,"and age is ", self.age)
obj1 = Student("Saket", 36)
obj1.age = 42          # update the age
obj1.method_func()

Delete Object Properties.
We can delete properties of an objects and an object also, by using the del keyword.
del obj1.age       # delete the properties (age) of an object.
del obj1              # Delete the obj1 object.

class Student:
def __init__(self, name, age):
self.name = name
self.age = age
def method_func(self):
print("Hi my name is " + self.name,"and age is ", self.age)
obj1 = Student("Saket", 36)
obj1.method_func()
del obj1.age
print(obj1.age)         # AttributeError: 'Student' object has no attribute 'age'.

The pass Statement.
class definitions cannot be empty, but for some reason the class definition with no content,
class Student:
pass                # put in the pass statement to avoid an error.
