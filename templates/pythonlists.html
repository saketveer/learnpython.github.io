Python Lists

There are four collection data types in the Python programming language:

• List 
• Tuple
• Set
• Dictionary

A list is a collection element which is ordered, changeable or mutable and allow duplicate values. Lists are used to store more than one items in a single variable. In Python lists are written with square brackets [].
  friends_list= ["Amit","Sumit","Rakesh","Rajesh","Saket","Saket"]

By using the list() constructor to make a List:
friends_list = list(("Amit","Sumit","Rakesh","Rajesh","Saket"))   # use the double rounded brackets.
print(friends_list)

List items are stored in  index, the first item has index [0], the second item has index [1] etc.

Access Lists Items.
friends= ["Amit","Sumit","Rakesh","Rajesh","Saket"]
print(friends)            # display all list's items in order
print(friends[0])       # display "Amit" which is first value of index
print(friends[-1])      # display "saket" the last value of index
print(friends[2:])      # display "Rakesh","Rajesh","Saket"
print(friends[2:4])    # it will return the items from zero index position 2 to 4th. while the item in position 4 is not included.

Note: some list methods that will change the order, while the order of the items will not change.

A list with strings, integers and Boolean values.
mix_list1 = ["pqrs", 40, True, 45, "female"]

Adding two list together.
My_numbers=[4,8,16,36,42,80]
friends= ["Amit","Sumit","Rakesh","Rajesh","Saket"]
friends.extend(My_numbers)
print(friends)    # result wil be ['Amit', 'Sumit', 'Rakesh', 'Rajesh', 'Saket', 4, 8, 16, 36, 42, 80]

What is the data type of a list?
friends= ["Amit","Sumit","Rakesh","Rajesh","Saket"]
print(type(friends))             # <class 'list'>

Append a new value by default it is in last index.
friends= ["Amit","Sumit","Rakesh","Rajesh","Saket"]
friends.append("Amrita")
print(friends)

Change the item name by index value.
friends= ["Amit","Sumit","Rakesh","Rajesh","Saket"]
friends[2] = "Mukesh"
print(friends)

Change a range of items.
friends= ["Amit","Sumit","Rakesh","Rajesh","Saket"]
friends[1:4] = ["Mukesh","Suresh"]
print(friends)

Loop in list.
friends= ["Amit","Sumit","Rakesh","Rajesh","Saket"]
for x in friends:
print(x)

Print all items by their index number.
friends= ["Amit","Sumit","Rakesh","Rajesh","Saket"]
for i in range(len(friends)):
print(friends [i])

Prints all item by using while loop.
friends= ["Amit","Sumit","Rakesh","Rajesh","Saket"]
i = 0
while i < len(friends):
print(friends [i])
i = i + 1

Looping Using List Comprehension / a short hand for loop that will print all items in a list.
friends= ["Amit","Sumit","Rakesh","Rajesh","Saket"]
[print(x) for x in friends]

List comprehension offers a shorter syntax to create a new list based on the values of an existing list.
friends= ["Amit","Sumit","Rakesh","Rajesh","Saket"]
mylist = []
for x in friends:
if "a" in x:
mylist.append(x)
print(mylist)

Add elements of a tuple to a list.
fruit_list = ["apple", "banana", "mango"]
fruit_tuple = ("guava", "orange")
fruit_list.extend(fruit_tuple)
print(fruit_list)

 Insert a new item.
friends= ["Amit","Sumit","Rakesh","Rajesh","Saket"]
friends.insert(4,"Amrita") # the new item added position 4 index
print(friends)

Find the value by index value.
friends= ["Amit","Sumit","Rakesh","Rajesh","Saket"]
friends.insert(6,"Amrita")
print(friends[5])
print(friends)

Remove a index value.
friends= ["Amit","Sumit","Rakesh","Rajesh","Saket"]
friends.remove("Sumit")
print(friends)

Clear the List index.
friends= ["Amit","Sumit","Rakesh","Rajesh","Saket"]
friends.clear()
print(friends)

Remove the last index value.
friends= ["Amit","Sumit","Rakesh","Rajesh","Saket"]
friends.pop()
print(friends)

Find the index number of value in list.
friends= ["Amit","Sumit","Rakesh","Rajesh","Saket"]
print(friends.index("Sumit"))

Count the value in list.
friends= ["Amit","Sumit","Rakesh","Rajesh","Saket","Sumit"]
print(friends.count("Sumit"))

Sort the value in alphabetical order in list.
friends= ["Amit","Sumit","Rakesh","Rajesh","Saket","Sumit","Binni"]
friends.sort()
print(friends)

Sort in reverse order the value order in list.
friends= ["Amit","Sumit","Rakesh","Rajesh","Saket","Sumit","Binni"]
friends.reverse()
print(friends)

Copy the list into same as another new list.
friends= ["Amit","Sumit","Rakesh","Rajesh","Saket","Sumit","Binni"]
students=friends.copy()
print(students)

Join two or more list together.
list1 = ["A", "B", "C"]
list2 = [1, 2, 3]
list3 = [4, 5, 6]
list3 = list1 + list2 + list3
print(list3)

List of Method & Description.
append()-- Adds an element at the end of the list
clear()------ Removes all the elements from the list
copy()------- Returns a copy of the list
count()----- Returns the number of elements with the specified value
extend()--- Add the elements of a list (or any iterable), to the end of the current list
index()-----  Returns the index of the first element with the specified value
insert()----- Adds an element at the specified position
pop()--------- Removes the element at the specified position
remove()--- Removes the item with the specified value
reverse()--- Reverses the order of the list
sort()--------- Sorts the list
