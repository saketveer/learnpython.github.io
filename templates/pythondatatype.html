Python-Data Types

Python has the following data types built-in by default.

Example Data Type
Numeric Types: int, float, complex
Text Type: str
Set Types: set, frozenset
Boolean Type: Bool
Binary Types: bytes, bytearray, memoryview


We can get the data type of any object by using the type()function: 
e.g. 
Find the data type of the variable x:
x = 5
print(type(x))

In Python, the data type is set when we assign a value to a variable:

Example Data Type
x = "Hello World" str
x = 40 int
x = 4.5 float
x = 2k complex
x = ["Mango", "Banana", "Orange"] list
x = ("Mango", "Banana", "Orange") tuple
x = range(10) range
x = {"name" : "Priya", "age" : 25} dict
x = {"Mango", "Banana", "Orange"} set
x = frozenset({"Mango", "Banana", "Orange"}) frozenset
x = False Bool
x = b"Hello" Bytes
x = bytearray(4) bytearray
x = memoryview(bytes(4)) memoryview

If we want to specify the data type, we can use the following constructor functions:

Example Data Type
x = str("Hello World") str
x = int(40) int
x = float(4.5) float
x = complex(2k)( complex
x = list(["Mango", "Banana", "Orange"]) list
x = tuple(("Mango", "Banana", "Orange")) tuple
x = range(10) range
x = dict({"name" : "Priya", "age" : 25}) dict
x = set({"Mango", "Banana", "Orange"}) set
x = frozenset({"Mango", "Banana", "Orange"}) frozenset
x = bool(8) Bool
x = bytes(8) Bytes
x = bytearray(8) bytearray
x = memoryview(bytes(8)) memoryview
