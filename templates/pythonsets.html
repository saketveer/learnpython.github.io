SET

SET is a collection element which is unindexed, unordered (output will be randomly), unchangeable or immutable but can be add new items as well as it does not allow duplicate values. SETs are used to store more than one items in a single variable. In Python sets are written with curly brackets {}.

Create a Set:
My_set = {"apple", "banana", "mango","guava"}
print(My_set)

The set() Constructor- It is also possible to use the set() constructor to make a set.
My_set = set(("apple", "banana", "cherry","mango","guava")) # note the double round-brackets
print(My_set)

Access Items
We cannot access items in a set by referring to an index, since sets are unordered the items has no index. But we can use for loop, or ask, if a specified value is present in a SET, by using the in keyword.

e.g. i
My_set = {"apple", "banana", "mango"}
for x in My_set:
print(x)

e.g. ii

My_set = {"apple", "banana", "mango"}
print("banana" in My_set)      # Check if "banana" is present in the set

Once a SET is created, we cannot change its items, but we can add new items.

Add Items to add one item to a set use the add() method.
To add more than one item to a set use the update() method.
My_set = {"apple", "banana", "cherry","mango"}
My_set.add("orange")
print(My_set)

My_set = {"apple", "banana", "cherry","mango"}
My_set.update(["orange", "mango", "grapes", "banana"])
print(My_set)     # duplicate items are not allowed.

Add Any Iterable
The object in the update() method does not have be a set, so it can be any iterable object (sets, tuples, lists, dictionaries).
My_set = {"apple", "banana", "cherry"}
Your_set = {"Guava-set",}
Our_tuple = ["Guava-tuple",]
This_list = ("Guava-list",)
My_set.update(Your_set)
My_set.update(Our_tuple)
My_set.update(This_list)
print(My_set)

Get the number of items in a set:
My_set = {"apple", "banana", "cherry", "mango"}
print(len(My_set))

To remove an item in a set, use the remove(), or the discard() method. If the item to remove does not exist in SET, remove() will raise an error while discard() not.

My_set = {"apple", "banana", "cherry","mango"}
My_set.remove("banana")
My_set.remove ("Tree")      # error
print(My_set)

My_set = {"apple", "banana", "cherry","mango"}
My_set.discard("apple")
My_set.discard("Tree")      # no error
print(My_set)

We can also use the pop(), method to remove an item, but this method will remove the last item. Remember that sets are unordered, so we will not know what item that gets removed. The return value of the pop() method is the removed item.
My_set = {"apple", "banana", "cherry","mango"}
print(My_set)
x = My_set.pop()
x = x + " is the last item is"
print(x)

The clear() method empties the SET.
My_set = {"apple", "banana", "cherry", "mango"}
My_set.clear()
print(My_set)

The del keyword will delete the set completely.
My_set = {"apple", "banana", "cherry","mango"}
del My_set
print(My_set)

Join two or more Sets- There are several ways to join two or more sets in Python. We can use the union() method that returns a new set containing all items from both sets, or the update() method that inserts all the items from one set into another.
Note: Both union() and update() will exclude any duplicate items.
My_set1 = {"p", "q" , "r"}
My_set2 = {1, 2, 3}
My_set3 = My_set1.union(My_set2)
print("after combined two sets \n")
print(My_set3)
My_set1 = {"a", "b" , "c"}
My_set2 = {1, 2, 3}
My_set1.update(My_set2)
print("after updating two sets \n")
print(My_set1)

The copy() method is used to copy another SET.
fruits = {"apple", "banana", "cherry","Guava"}
x = fruits.copy()
print(fruits)
print("after copied \n")
print(x)

The difference() method returns a set containing the difference between two or more sets.
x = {"apple", "banana", "cherry","mango"}
y = {"google", "microsoft", "apple","guava"}
z = x.difference(y)
print(z)
