Python If Else

Python supports the usual logical conditions from mathematics:
• Greater than: a > b
• Less than: a < b
• Greater than or equal to: a >= b
• Less than or equal to: a <= b
• Equals: a == b
• Not Equals: a != b
These conditions can be used commonly in "if statements" and loops.

An if statement is written by using the ''if'' keyword.
p = 50              # p is a integer
q = 100           # q is a integer
if q > p:
 print("q is greater than p")

Indentation
Python relies on indentation (whitespace at the beginning of a line) to define scope in the code.

Elif
The elif keyword is used when if the previous conditions were not true.
p = 100
q = 50
if q > p:
 print("q is greater than p")
elif p > q:
 print("p is greater than q")

Else
The else keyword catches anything which isn't caught by the preceding conditions.
p = 100
q = 50
if q > p:
 print("q is greater than p")
elif p == q:
 print("p and q is equal")
else:
 print("p is greater than q") # because if & elif condition false, else will be execute.

The else can be used without the elif:
p = 100
q = 50
if q > p:
 print("q is greater than p")
else:
 print("p is greater than q")

Short Hand If & Else
Only one statement to execute, we can put it on the same line as the if statement.

p = 100
q = 10
if p > q: print("p is greater than q")             # example 1
print("Good") if p > q else print("Bad")      # example 2


This is known as Ternary Operators, or Conditional Expressions.

p = 20
q = 20
print("p is grater") if p > q else print("=") if p == q else print("q is grater")

And / ( And Gate)
The "and" keyword is a logical operator, and it only care that both conditions are true.
p = 100
q = 30
r = 200
if p > q and r > p:
print("Both conditions are True")

Or / (Or Gate)
The "or" keyword is a logical operator, and it only care that at least one condition is true.
p = 100
q = 30
r = 200
if p > q or p > r:
print("Only one condition is True")

Nested If
The "if" statements inside another "if" statements, it is called "nested if" statements.
p = 200
if p > 60:
print("p is greater than 60,")
if p > 150:
print("as well as greater than 150!")
else:
print("but p is less than 150!")
else:
print("p is less than 60 & 150!")

The pass Statement
For some reason has "if" statement with no content, put in the "pass" statement to avoid getting an error.
p = 100
q = 200

if p > q:
pass      # because no content.
