Python Tuple

Tuple

It is a collection element which is ordered, unchangeable or immutable and allow duplicate values. Tuples are used to store more than one items in a single variable. In Python tuples are written with round brackets ().
friends_tuple= (“Amit”,”Sumit”,”Rakesh”,”Rajesh”,”Saket”,”Saket”)

Ordered:
The tuples are ordered, the items have a defined order, and that order will not change.

Unchangeable:
Tuples are unchangeable or immutable , we cannot change, add or remove items after the tuple has been created.

Allow Duplicates:
Tuples are stored item in index, tuples can have items with the same value:

Create a Tuple:
My_tuple = ("apple", "banana", "orange")
print(My_tuple)
print(My_tuple[1])

Tuple with duplicate value:
My_tuple = ("apple", "banana", "orange", "banana")
print(My_tuple)

Find the number of items in the tuple:
My_tuple = ("apple", "banana", "orange", "banana")
print(len(My_tuple))

string, int and boolean data types:
My_tuple1 = ("apple", "banana", "orange", "banana")
My_tuple2 = (1, 2, 3, 4, 100)
My_tuple3 = (False, True, False)
print(My_tuple1)
print(My_tuple2)
print(My_tuple3)

A tuple with strings, integers and boolean values:
My_tuple=("ABC", 45, True, False, 50, "male")
print(My_tuple)

What is the data type of a Tuple?
friends= (“Amit”,”Sumit”,”Rakesh”,”Rajesh”,”Saket”)
print(type(friends))        # <class ‘Tuple’>

By using the tuple() constructor to make a tuple:
friends_tuple = tuple((“Amit”,”Sumit”,”Rakesh”,”Rajesh”,”Saket”)) # use the double rounded brackets ().
print(friends_tuple)

We can access tuple items by referring to the index number, inside square brackets:
My_tuple = ("apple", "banana", "orange")
print(My_tuple[1])

Print the last item of the tuple: 
This_tuple = ("apple", "banana", "cherry")
print(This_tuple[-1])

# Negative indexing means beginning from the end, -1 refers to the last item, -2 refers to the second last item etc.

Range of Indexes:
My_tuple = ("apple", "banana", "cherry", "orange", "kiwi", "melon", "mango")
print(My_tuple[2:5])
print(My_tuple[:5])
print(My_tuple[2:])

Range of Negative Indexes:
This example returns the items from index -4 (included) to index -1 (excluded).
This_tuple = ("apple", "banana", "cherry", "orange", "kiwi", "melon", "mango")
print(This_tuple[-4:-1])

Join two or more tuples by useing the + operator:
tuple1 = ("a", "b", "c")
tuple2 = (1, 2, 3)
tuple3 = tuple1 + tuple2
print(tuple3)

Multiply the fruits tuple by 3:
fruits = ("apple", "banana", "mango")
result_tuple = fruits * 3
print(result_tuple)

Change Tuple Values Once a tuple is created, We cannot change its values. Tuples are unchangeable, or immutable. But at run time. We can convert the tuple into a list, change value, and convert the list back into a tuple.
x = ("apple", "banana", "cherry")
y = list(x)
y[1] = "kiwi"
x = tuple(y)
print(x)

By using for Loop or Iterate through the items and print the values:
This_tuple = ("apple", "banana", "cherry")
for x in This_tuple:
print(x)

Check "if " cherry is present in the tuple:
This_tuple = ("apple", "banana", "cherry", "orange")
if "cherry" in This_tuple:
print("Yes, 'item' is in the fruits tuple")

Tuple Length to determine how many items a tuple has, use the len() method:
This_tuple = ("apple", "banana", "cherry", "orange")
print(len(This_tuple))

Create a Tuple with only one item, We have to add a comma after the item, otherwise Python will not recognize it as a tuple.
This_tuple = ("apple",)
print(type(This_tuple))

#NOT a tuple
This_tuple = ("apple")
print(type(This_tuple))

Remove Items from tuple. We cannot remove items in a tuple. Tuples are unchangeable, so We have to delete the tuple completely. By using del keyword we can delete the tuple completely.
This_tuple = ("apple", "banana", "orange","mango")
del This_tuple
print(This_tuple)    # it will raise an error because the tuple no longer exists

Unpacking a Tuple when tuple is created, normally assign values to it. It is called "packing" While extracting value one-by one is called unpacking.
fruit_tuple = ("apple", "banana", "orange","mango")
(apple1, banana1, orange1, mango1) = fruit_tuple
print(apple1)
print(banana1)
print(orange1)
print(mango1)
print(apple1) # result repeated

Note: The number of variables must match the number of values in the tuple.

Using Asterix* If the number of variables is less than the number of values of the tuple, we can add an * to the variable name and the values will be assigned to the variable as a list.
fruit_tuple = ("apple", "banana", "orange","mango")
(apple1, banana1, *orange1) = fruit_tuple
print(apple1)
print(banana1)
print(orange1) # rest of values assigned to orange1.

If the Asterix* is added to another variable name Python will assign all values to that variable and rest of values assigned to another respective variable.
fruit_tuple = ("apple", "mango", "papaya", "pineapple", "cherry", "bherry")
(*apple1, mango1, papaya1) = fruit_tuple
print(apple1)
print(mango1)
print(papaya1)

Join two or more tuples by using the + operator:
My_tuple1 = ("P", "Q" , "R")
My_tuple2 = (6, 7, 8)
My_tuple3 = (1, 2, 3)
My_tuple4 = My_tuple1 + My_tuple2 + My_tuple3
print(My_tuple4)

The tuple() Constructor can be use make a tuple.
This_tuple = tuple(("apple", "banana", "mango")) # note the double round-brackets
print(This_tuple)

Count the no. of items in a tuple:
This_tuple = (1, 3, 7, 8, 7, 5, 4, 6, 8, 5)
x = This_tuple.count(5)
print(x)

Find the index position of an item:
This_tuple = (1, 3, 7, 8, 7, 5, 4, 6, 8, 5)
x = This_tuple.index(4)
print(x)
